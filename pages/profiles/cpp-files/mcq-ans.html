Answers with Explanations - C++ MCQ

  1. Correct Answer: B. int *ptr;
    Explanation: In C++, a pointer is declared using the asterisk symbol `*` before the variable name. For example, `int *ptr;` means that `ptr` is a pointer to an integer.

  2. Correct Answer: B. private
    Explanation: By default, all members of a class in C++ are `private`. This means they are not accessible outside the class unless specified otherwise.

  3. Correct Answer: D. : (colon)
    Explanation: In C++, inheritance is specified using a colon followed by the access specifier and the base class name. Example: `class B : public A {}`.

  4. Correct Answer: B. 2
    Explanation: In C++, dividing two integers results in integer division. `5 / 2` equals `2` (decimal part is discarded).

  5. Correct Answer: C. real
    Explanation: `float`, `double`, and `char` are valid C++ data types. `real` is not a standard data type in C++.

  6. Correct Answer: B. 4 bytes
    Explanation: On a typical 32-bit system, an `int` occupies 4 bytes of memory.

  7. Correct Answer: C. new
    Explanation: C++ uses the `new` operator to dynamically allocate memory at runtime. Example: `int* p = new int;`.

  8. Correct Answer: B. // comment
    Explanation: Single-line comments in C++ begin with `//`. Multi-line comments use `/* ... */`.

  9. Correct Answer: B. Current object
    Explanation: The `this` pointer in C++ is an implicit pointer that refers to the current object of a class.

  10. Correct Answer: B. 10
    Explanation: The value of `b` (10) is assigned to `a`. Then `b` is changed to 20, but `a` still holds the previous value (10).
