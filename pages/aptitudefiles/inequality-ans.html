Ans1. Answer: Only conclusion is true
Explanation: Given: A > B = C ≥ D < E
From this: A > B = C ≥ D → A > D is definitely true.

Ans2. Answer: Cannot be determined
Explanation: P ≥ Q > R ≤ S < T. The relation between P and S is not clear, since R ≤ S and R is less than Q. So P> S cannot be concluded definitely.

Ans3. Answer: Definitely true
Explanation: M < N=O ≥ P> Q
From this: N = O ≥ P > Q → N > Q is definitely true.

Ans4. Answer: True
Explanation: W = X ≥ Y > Z
Since W = X and X ≥ Y > Z → So W > Z is definitely true ⇒ Z < W is true.
